## Ingredient Image Strategy

For EVERY shot, upload:
1. **Primary:** The scene still for that specific shot
2. **Secondary:** Character Full Body Turnaround (for character consistency)

For shots in the server room (6.2, 2.2, 6.1, 7.1), optionally add:
3. **Tertiary:** Environment/Colour Reference (for lighting consistency)

## Scene Extension Candidates

If any shot needs to be longer than 8 seconds in the final edit:
- Use Veo 3.1 Scene Extension
- Screenshot the final frame of the 8-second clip
- Feed that frame as the ingredient for the next 8-second extension
- This creates seamless 16-second continuous shots

**Best candidates for extension:**
- Shot 3.1 (Tor Nodes) — the slow pan could benefit from 16 seconds
- Shot 8.2 (Close) — the slow pull-back could be extended for dramatic effect
